# Knouz ERP (نظام إدارة أعمال كنوز الإنجاز)

نظام متكامل لإدارة الأعمال (ERP & CRM) مبني باستخدام لغة PHP وقواعد بيانات MySQL. تم تصميم النظام لتلبية احتياجات إدارة العملاء، الطلبات، الفواتير، ومتابعة العمليات اليومية بكفاءة عالية مع واجهة مستخدم حديثة وسهلة الاستخدام تدعم اللغة العربية بالكامل (RTL).

## 🚀 المميزات الرئيسية

* **إدارة العملاء والشركات:** سجلات شاملة للعملاء (360 درجة)، تشمل كافة التفاصيل وعناوينهم والشركات التابعة لهم.
* **إدارة الطلبات والخدمات:** إنشاء وتتبع طلبات الخدمات، تحديث حالات الطلب بشكل مستمر.
* **إدارة المالية:** إصدار عروض الأسعار (Quotations)، الفواتير (Invoices)، وتتبع المدفوعات (Payments) مع حساب تلقائي للضريبة وتفقيط المبالغ.
* **إدارة الفرص البيعية (CRM):** تتبع الفرص البيعية، تسجيل المكالمات الصادرة والواردة، ومتابعة التواصل مع العملاء.
* **إدارة المستندات وتتبع التجديدات:** مركز التجديدات لمتابعة السجلات والوثائق التي تقترب من الانتهاء، ومساحة لرفع وإدارة المستندات.
* **تقارير وإحصائيات:** لوحة تحكم تفاعلية وتقارير شاملة (مالية، خدمات، عملاء).
* **إدارة المستخدمين والصلاحيات (RBAC):** نظام صلاحيات متقدم يضمن أمان البيانات والوصول المحدد لكل دور وظيفي.

## 🛠 التقنيات المستخدمة

* **Backend:** PHP 8.x (Pure / Custom MVC Architecture)
* **Database:** MySQL
* **Frontend:** HTML5, CSS3 (Vanilla, Custom UI System), JavaScript
* **Icons:** FontAwesome

## ⚙️ متطلبات التشغيل

* خادم ويب يدعم PHP (مثل Apache عبر XAMPP/WAMP أو Nginx).
* إصدار PHP 8.0 أو أعلى (يُفضل 8.1+).
* قاعدة بيانات MySQL أو MariaDB.

## 📥 التثبيت والإعداد

1. **نسخ المشروع:**
   قم بنسخ المستودع إلى مجلد الخادم المحلي (مثلاً `htdocs` في XAMPP):
   ```bash
   git clone https://github.com/mustafa-ishag/knouz-erp.git kn
   ```

2. **إعداد قاعدة البيانات:**
   * قم بإنشاء قاعدة بيانات جديدة في MySQL باسم `knouz_db`.
   * قم باستيراد الجداول من ملف الهيكل المتواجد في `database/schema.sql`.
   * (اختياري) قم باستيراد البيانات الأولية/الأساسية من ملف `database/seed.sql`.

3. **إعداد الاتصال بقاعدة البيانات:**
   * تأكد من إعدادات الاتصال في ملف `config/database.php` لتتطابق مع خادمك (اسم المستخدم وكلمة المرور).

4. **تشغيل النظام:**
   * افتح المتصفح وتوجه إلى مسار المشروع، على سبيل المثال: `http://localhost/kn/public/`
   * يمكنك تسجيل الدخول باستخدام حساب المشرف الافتراضي الذي قمت بإنشائه أثناء إعداد قاعدة البيانات.

## 📂 هيكلية المشروع الأساسية

* `config/`: إعدادات النظام وقاعدة البيانات.
* `core/`: قلب النظام ويحتوي على الكلاسات الأساسية (Router, Model, Controller, Auth, Database).
* `domain/`: وحدات النظام (Modules) مقسمة بشكل نظيف (عملاء، فواتير، مدفوعات...الخ).
* `public/`: نقطة الدخول للنظام (index.php) والملفات العامة (CSS, JS, Uploads).
* `templates/`: القوالب الأساسية للواجهة (Header, Sidebar, Layout).
